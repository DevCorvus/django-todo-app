Hi! my name is Luis Portillo, Venezuelan Fullstack Web Developer (DevCorvus). This time I have decided to do this learning project in Django, something quite simple but in order to explore its features, development philosophy among other things, in addition to being based on one of my favorite languages (Python).

This is a simple task app with groups and messages, but internally it covers many of the essential features that Django offers.

The development was relatively simple and fast but it required an extensive reading of documentation especially to customize various built-in functionalities. Many of which were relatively uncomfortable at first, but once you get the hang of it, it's pretty easy and comfortable.

Clearly this is a learning project and it lacks many functionalities since the purpose was mainly to test the framework and what it had to offer. There is an API endpoint in /api/ just to test the Django Rest Framework, but only that as it is generally redundant in terms of my learning.
